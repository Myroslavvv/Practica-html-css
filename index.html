Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

Git iseasy to learn and has a tiny footprint with lighting fast performance. It outclasse SCM tools like Subversion, CVS, Perforce, and ClearCas with features like cheap local branching, convenient staging areas, and multiple workflows.
